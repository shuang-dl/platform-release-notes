Summary

This release ships 2 changes, both in PM Operations. Owner records gain optional custom fields — a drawer overview card and list-engine filters, with value columns still pending a server endpoint — but it ships behind the custom-fields-owner flag, so Help Center updates should wait for GA. The Maintenance Dashboard gets an AI view toggle, refreshed filters, and clickable Sankey drill-downs, continuing the same dashboard epic already flagged as an undocumented gap in v9.4.20, v9.4.22, and v9.4.32 — this item folds into that same suggested article rather than needing a new one.
