Summary

This release ships 4 changes across PM Operations (3) and Leads & Leasing (1). The headline customer-facing feature adds a Service Location callout on work order details and printouts for units with distinct addresses — helping vendors arrive at the right location. A SEV-2 fix restores the rental applications listings page for companies whose units have no rent set. Two internal fixes resolve a global search note-click crash and suppress voice call error log noise.

PMO-358 Service Location on Work Orders Added a Service Location callout on work order details and printouts when a unit has its own address. When a work order is linked to a unit whose address differs from the property's (e.g. multi-building), a light-blue "Service Location" callout now appears on the work order detail panel and print/PDF so vendors and assigned users see the correct unit address.
