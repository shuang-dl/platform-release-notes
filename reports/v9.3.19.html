PMO-1553 Communications Center — secure-link fallback for oversized attachments
🟣 BETA flag: commsCenterLinkFallback 		Now supports sending oversized or unsupported attachments as secure links in text messages instead of failing. When a PM attaches a file to an outbound SMS/MMS in Communications Center, DoorLoop now checks at attach-time whether the file exceeds MMS limits (5 MB) or is an unsupported type. If it does, the system compresses first and — when compression alone isn't enough — hosts the file on S3 and inserts a short secure link in the message body instead. The sent message shows the native attachment view plus a "Sent as link" badge. Recipients receive a no-login tokenized link. Feature is gated behind commsCenterLinkFallback (default off, cohort ramp). BETA — hold for GA

L2L-1611 Property timezone auto-detection (create & edit forms)
🟣 BETA flag: property-timezone (off) 		Added property timezone selection that auto-fills from address coordinates and updates when they change. Adds a TimeZone selector to the property create wizard and address-edit form, gated behind the property-timezone flag (default off). When a user enters or changes an address, the selector auto-derives the IANA timezone from the coordinates via a new read-only endpoint and pre-fills the field — which the user can still override. The submitted timezone is stored on address.timezone. Primarily powers Showings availability accuracy.
