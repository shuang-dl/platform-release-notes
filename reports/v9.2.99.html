PMO-968 Per-Property Welcome SMS Customization in Communication Center Added company-level Welcome SMS customization in Communication Center, with default fallback and required opt-out text. Property managers can now customize the Welcome SMS message per property in the Communication Center settings. A new tab in the settings dialog accepts property-specific welcome text using template variables ({{companyName}}, {{fromNumber}}, {{recipientName}}). When no property-level message is set the system falls back to the company-wide default. The required opt-out language is enforced automatically. Addresses a high-priority request from large upmarket accounts (~450–641 units) managing multiple owners or LLCs who need each property's outbound SMS to reflect the correct business name. MATCHED
