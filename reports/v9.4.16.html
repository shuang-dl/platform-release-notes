Accounting 1 item

Linear Feature Original Notes (Slack) What it does Affected articles Affected videos
TEN-960 Feature-Flag Evaluation Telemetry (Internal) Added telemetry for feature-flag evaluations to help spot slow, errored, or defaulted checks without changing results. Adds a backend OpenFeature observability hook that records per-evaluation timing, reason, error code, and event-loop utilization to a New Relic custom event/dashboard, so engineers can separate real flag-eval cost from event-loop queue-wait and catch flags silently falling to default. Purely observational — no change to what any flag evaluates to or how the app behaves for users. INTERNAL — backend telemetry/observability hook only; no user-facing behavior or UI change, so no Help Center update is needed. — no related video

L2L-1753 Fee Collection Column for Rental Applications Added a Fee collection column to rental application lists and settings, showing deposit and fee totals when enabled. Adds a "Fee collection" column to the Rental Application list and Settings screens that totals everything a landlord will collect: the sum of all configured fee line items, plus the fee-revenue portion of any Customizable Plan charge, plus the sum of every deposit type configured (with no double-counting when multiple deposits exist).
