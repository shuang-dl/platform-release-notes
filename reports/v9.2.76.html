Release v9.2.76 — 2026-06-03

View original Slack post · 1780494111.566169

Summary

This release ships 22 changes across 7 areas. Customer-visible highlights include a redesigned vendor portal with messaging and file uploads (PMO-221), AI Communications Center guided setup and escalation notifications (PMO-1493), rental applications in global search (WOR-822), and corrected transfer notification wording (BAN-789). Three features are in beta behind feature flags: late-balance-notice workflows (WOR-430/432) and a new rental application decision dialog (L2L-1151). The remaining 11 items are internal infrastructure, monitoring, and performance fixes.
